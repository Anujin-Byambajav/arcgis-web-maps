[This](https://csusb.maps.arcgis.com/home/item.html?id=d03336a264cb4854ab9ee14aa8304eef) interactive web map displays the locations of earthquakes that have occurred around the world since 1970. The data for this map is sourced from the United States Geological Survey (USGS) earthquake catalog, which is regularly updated to include the latest seismic events.
